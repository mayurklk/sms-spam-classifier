# SMS Spam Classifier

This project is a Machine Learning based SMS Spam Classifier that predicts whether an SMS message is Spam or Not Spam (Ham) using Natural Language Processing techniques.

The model is trained in a Jupyter Notebook and deployed using a Python web application.

---

## Project Overview

- Text preprocessing and cleaning
- Feature extraction using TF-IDF Vectorizer
- Machine learning based text classification
- Simple web application for prediction

---

## How to Run the Project

### Step 2: Navigate to the project directory

### Step 4: Install required dependencies

### Step 3: Create a virtual environment (optional but recommended)

### Step 5: Run the application

### Step 6: Open the application in browser
